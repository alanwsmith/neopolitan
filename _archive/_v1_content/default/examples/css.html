Section: css

The -> css section is allows for arbitrary CSS to be added to a page. Each section adds a <style> element just before the closing </head> tag in the HTML. Multiple sections output from the top down in the order they appear in the source file.

Source
-> css

pre {
  border: 2px solid red;
}
