Neopolitan

Inline Tag: kbd (The Keyboard Input Element)

Basic Usage

Source
-> p

Use a <<pencil|kbd>> to write
Output
<p>Use a <kbd>pencil</kbd>  to write</p>

With Attributes

Source
-> p

Use a <<pencil|kbd|id: alfa|class: bravo>> to write
Output
<p>Use a <kbd id="alfa" class="bravo">pencil</kbd>  to write</p>
