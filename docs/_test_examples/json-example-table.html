JSON Table Example

From JSON To Table

This is a basic example showing how JSON can be used to create tables. It can be used directly or as a starting point to make tables more specific to your needs:

NOTE: the format for the JSON is still under development and may change.

NOTE: This is mainly an example to show JSON usage. If you're looking for a table in lots of cases just assembling the HTML and dropping it in a -- html block might make more sense.

NOTE: You could also make a much simpler table that is just arrays of arrays for the data
