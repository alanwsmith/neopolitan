Shorthands

Introduction

Shorthands offer a way to enhance content in Neopolitan files1.

@ = wraps in span tag to make it quicker to
add attributes and flags. 

` = code

* = strong

_ = emphasis

^ = footnote

~ = delete 

( and ) = insert

[ and ] = link

{ and } = image
