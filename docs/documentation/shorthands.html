Tags and Shorthands

Introduction

Neopolitan provides two ways to enhance content inside of sections:

Tags

< and > = named tag

` = code

* = strong

_ = emphasis

[ and ] = link

^ = footnote

~ = delete

( and ) = insert

{ and } = image

@ = wraps in span tag to make it quicker to add attributes and flags.
