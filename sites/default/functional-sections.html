Functional Sections

Implemented

- comment - Section that's not output

- css - A place to put CSS that will be inlined in the page

- html - for raw html blocks that go directly to the output without further processing

- neoexample - A display for examples on the neopolitan documentation site

In Progress

- ext - wip Call to an external process

- head - wip raw content to put in the head of the document

- include - wip - For putting the content of one file inside another

- script - A raw section that adds script tags to the header of the page

- widget - wip An area to pass data to the client or build processes for custom functionality
