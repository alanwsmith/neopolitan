Links and Link References

This is still a draft

Links are done with one of the following options

# not yet implemend
>text>URL>

# not yet implemend
<<l|TEXT|URL>>

# not yet implemend
<<l|TEXT|URL|KEY: VALUE|KEY: ETC...>>

# not yet implemend
<<link|TEXT|URL>>

# not yet implemend
<<link|TEXT|URL|KEY: VALUE|KEY: ETC...>>

The links can span multiple lines like this:

>text>
https://www.example.com/>

or

<<link|
https://www.example.com/|
key: value|
key: value>>

Reference Links

The links don't have to have the URL in them directly, they can be reference to links stored in a -> links section

<<l|some text|ref_name>>

-> links

- ref_name:

As long as it mathces it'll be used for the link.
