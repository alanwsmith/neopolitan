Section: warning

  • Convenience for a <div> with a warning class on it

Source
-> warning

Jazz and swing fans like fast music

The map had an X that meant nothing
Output
<div class="warning">
  <p>Jazz and swing fans like fast music</p>
  <p>The map had an X that meant nothing</p>
</div>

  • Attributes can be added

Source
-> warning
>> id: zulu

Tear a thin sheet from the yellow pad

Split the log with a quick, sharp blow
Output
<div class="warning" id="zulu">
  <p>Tear a thin sheet from the yellow pad</p>
  <p>Split the log with a quick, sharp blow</p>
</div>

  • Classes are added to the existing warning class

Source
-> warning
>> class: alfa

The box was thrown beside the parked truck

The frosty air passed through the coat
Output
<div class="warning alfa">
  <p>The box was thrown beside the parked truck</p>
  <p>The frosty air passed through the coat</p>
</div>
