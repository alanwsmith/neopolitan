Section: image

A section that uses attributes for the main image call and uses the content area for the alt text.

The first attribute line is used to set the src path of the image by default:

Source
-> image
>> alfa.jpg

Example alfa
Output
<img src="alfa.jpg" alt="Example alfa">

Other attributes can be passed as well:

Source
-> image
>> bravo.jpg
>> id: img_bravo 
>> class: charlie 

Example bravo
Output
<img src="bravo.jpg" id="img_bravo " class="charlie " alt="Example bravo">

It's also possible to use src: path.jpg to set the path. If you do that the first line must have a key as well as a value. For example:

Source
-> image
>> id: img_delta
>> src: delta.jpg

Example delta
Output
<img id="img_delta" src="delta.jpg" alt="Example delta">
