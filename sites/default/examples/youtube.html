Section: youtube

  • Convenience method for a YouTube embed wrapped in a div with a class of youtubeVideo

  • The embed code is taken directly from the YouTube share button

  • The first >> attribute line is the ID of the video for the embed

Source
-> youtube
>> uh9i3be2wIE
Output
<div class="youtubeVideo" uh9i3be2wIE>
  <iframe 
    src="https://www.youtube.com/embed/uh9i3be2wIE"
    width="560" height="315" 
    title="YouTube video player" frameborder="0"
    allow="accelerometer; autoplay; clipboard-write; 
    encrypted-media; gyroscope; picture-in-picture; web-share"
    allowfullscreen></iframe>
</div>

  • Attributes can be added

Source
-> youtube
>> gnUYoQ1pwes
>> id: victor
Output
<div class="youtubeVideo" gnUYoQ1pwes id="victor">
  <iframe 
    src="https://www.youtube.com/embed/gnUYoQ1pwes"
    width="560" height="315" 
    title="YouTube video player" frameborder="0"
    allow="accelerometer; autoplay; clipboard-write; 
    encrypted-media; gyroscope; picture-in-picture; web-share"
    allowfullscreen></iframe>
</div>

  • If a class is added it gets appended to the existing title class

Source
-> youtube
>> HgzGwKwLmgM
>> class: queen
Output
<div class="youtubeVideo queen" HgzGwKwLmgM>
  <iframe 
    src="https://www.youtube.com/embed/HgzGwKwLmgM"
    width="560" height="315" 
    title="YouTube video player" frameborder="0"
    allow="accelerometer; autoplay; clipboard-write; 
    encrypted-media; gyroscope; picture-in-picture; web-share"
    allowfullscreen></iframe>
</div>
