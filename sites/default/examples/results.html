results section

The idea with this one is to use it after code blocks to show their corresponding output.

I considered using the <samp> tag for my implementation but code highlighters are generally geared for <pre><code> blocks.

Source
-> results

The colt reared and threw the tall rider.
The wide road shimmered in the hot sun.
Output
<pre class="results"><code>The colt reared and threw the tall rider.
The wide road shimmered in the hot sun.</code></pre>
Source
-> results
>> id: sierra
>> class: tango

Fish evade the net.
Five years with a shaggy dog.
Output
<pre class="results tango" id="sierra"><code>Fish evade the net.
Five years with a shaggy dog.</code></pre>
