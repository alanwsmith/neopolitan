code section

These are code blocks similar to code fence blocks in Markdown. If the first attribute has no value it is treated as the language and added as a language- class to the <pre> element

-> code

console.log("Hello")
console.log(", ")
console.log("World")

<pre><code>console.log(&quot;Hello&quot;)
console.log(&quot;, &quot;)
console.log(&quot;World&quot;)</code></pre>
