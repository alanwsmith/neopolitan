Section Attributes

Overview

  • Every section accepts attributes

  • Attributes are defined on lines starting with >> directly after the section line starting with ->. - Keys are separated from values with a : character

  • The keys and values are output directly to the primary HTML tag for the section. For example:

Source
-> h4
>> id: alfa
>> data-widget: bravo

Open the crate
Output
<h4 id="alfa" data-widget="bravo">Open the crate</h4>

  • If there is no value, the key is passed by itself

Source
-> h4
>> hidden

Pile the coal
Output
<h4 hidden>Pile the coal</h4>

Special Handling Shortcuts

  • The code, image, vimeo, and youtube sections treat the first attribute line in a special manner if there's no key

  • The code section uses a key only first attribute line to define the language of the section

Source
-> code
>> rust

println!("Hello, World");
Output
<div class="pre-wrapper">
  <div class="pre-spacer"></div>
  <pre><code class="language-rust">println!(&quot;Hello, World&quot;);</code></pre>
</div>

  • image sections use the first item as the src path

Source
-> image
>> https://placekitten.com/200/300

A cute kitten
Output
<img src="https://placekitten.com/200/300" alt="A cute kitten">

  • The youtube and vimeo sections use the first line as the ID to embed:

Source
-> youtube
>> gnUYoQ1pwes
Output
<div class="youtubeVideo" gnUYoQ1pwes>
  <iframe 
    src="https://www.youtube.com/embed/gnUYoQ1pwes"
    width="560" height="315" 
    title="YouTube video player" frameborder="0"
    allow="accelerometer; autoplay; clipboard-write; 
    encrypted-media; gyroscope; picture-in-picture; web-share"
    allowfullscreen></iframe>
</div>

Additioal Classes

  • Several sections (note, notes, results, subtitle, title, todo, vimeo, warning, youtube) have predefined HTML classes in their attributes - Passing a key/value pair with a class in the section header adds the class to the ones that's already there

Source
-> note
>> class: echo

Take the winding path
Output
<div class="note echo">
  <div class="noteline">note</div>
  <div class="notetext"><p>Take the winding path</p></div>
</div>
