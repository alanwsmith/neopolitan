Neopolitan Unit Tests

This document is use to verify neopolitan functionality.

Details

  • Each CONTENT section that's followed by two PRE sections collectively constitute a test
  • The CONTENT section contains a description of the test
  • The first PRE section after the CONTENT section contains the input for the test
  • The second PRE section contains the expected result sandwiched between rows of dashes above and below to avoid whitespace variations
  • In all cases in this file, the content represents that coming from a CONTENT sections (i.e. not a PRE, HTML, etc...)
  • The other sections the same processing applies to are DIVs. TBD on others
