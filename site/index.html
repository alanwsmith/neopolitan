Section Types

  • aside - The HTML <aside> tag

  • attributes - A collection of key, value metadata pairs for the page. I use it for the ID, date, status, etc... The permitted and parsed values are left as an implementaiton detail. My implementation is straight attirbutes like the rest of the sections but YAML or other formats could be used with a corresponding implementation engine.

  • blockquote - The HTML <blockquote> tag

  • blurb - Not rendered on the page. Used as meta data for links into the page. - code - Code blocks like fenced code blocks but you can add an arbitrary attributes to them. - comment - Parsed like text sections in the AST but with the expectation that they won't output unless the implementation is specifically designed to do so. - css - A place to put CSS that will be inlined in the page

  • div - A <div> wrapper that can be used to add attributes to sections of content
