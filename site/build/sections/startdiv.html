Section: div

This section provides a div wrapper around the rest of the content the gets processed as normal. It allows for adding attributes to that div as well.

Source
-> startdiv
>> id: alfa
>> class: bravo


-> list

- Turn the lantern

- Raise the sail

-> enddiv
Output
<div id="alfa" class="bravo"></div><ul>
<li><p>Turn the lantern</p></li>
<li><p>Raise the sail</p></li>
</ul>

Solve the issue where two blank lines have to be entered after the `-> enddiv`` tag. (Same thing after the `-> startdiv``)
