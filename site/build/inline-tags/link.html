Neopolitan

Inline Tag: link (The External Resource Link Element)

Basic Usage

Source
-> p

Use a <<pencil|link|https://www.example.com/>> to write
Output
<p>Use a <a href="https://www.example.com/">pencil</a>  to write</p>

With Attributes

Source
-> p

Use a <<pencil|link|https://www.example.com/|class: echo>> to write
Output
<p>Use a <a href="https://www.example.com/" class="echo">pencil</a>  to write</p>
