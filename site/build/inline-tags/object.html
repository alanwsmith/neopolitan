Neopolitan

Inline Tag: object (The External Object Element)

Basic Usage

Source
-> p

Use a <<pencil|object>> to write
Output
<p>Use a <object>pencil</object>  to write</p>

With Attributes

Source
-> p

Use a <<pencil|object|id: alfa|class: bravo>> to write
Output
<p>Use a <object id="alfa" class="bravo">pencil</object>  to write</p>
