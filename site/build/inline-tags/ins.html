Neopolitan

Inline Tag: ins (The Insert Element)

Basic Usage

Source
-> p

Use a <<pencil|ins>> to write
Output
<p>Use a <ins>pencil</ins>  to write</p>

With Attributes

Source
-> p

Use a <<pencil|ins|id: alfa|class: bravo>> to write
Output
<p>Use a <ins id="alfa" class="bravo">pencil</ins>  to write</p>
