blockquote section

For the HTML <blockquote> tag.

Example Without Attributes

-> blockquote 

A blockquote without attributes

Results

<blockquote>
  <p>A blockquote without attributes</p>
</blockquote>

Example with attributes

-> blockquote
>> cite: https://www.example.com/

A blockquote with attributes

Results

<blockquote cite="https://www.example.com">
  <p>A blockquote with attributes</p>
</blockquote>

Output

A blockquote with attributes
