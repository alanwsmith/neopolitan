results section

The idea with this one is to use it after code blocks to show their corresponding output. I considered using the <samp> tag for my implementation but code highlighters are generally geared for <pre><code> blocks.

Example

-> results

The colt reared and threw the tall rider.
The wide road shimmered in the hot sun.
<pre class="results"><code>
The colt reared and threw the tall rider.
The wide road shimmered in the hot sun.
</code></pre>
